# Krank. Small platformer game  
demo link https://iviv122.itch.io/krank 

Was made with Unity 6000.0.45f1 (stable)

## Game Controlls

Use WASD to move
Use Mouse to look around 
Use Space to jump
Use Left Mouse Button to pick up
Hold Left Mouse Button to throw
